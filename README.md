### 1. `CFG (code-first-girls)`
**Python and Apps course sponsored by Barclays**  
This project is a Python implementation of a **Top Trumps game using the Pokemon API**.  

**Project Highlights:**
- Players are randomly allocated a Pokemon and compare chosen stats to determine the winner.  
- Allows customization of the **number of rounds**.  
- Players can select a **stat** to compare each round.  
- Winner of each round is determined automatically by the **highest stat value**.  

**Extensions & Features:**
- **Additional Characteristics:** Added more attributes from the Pokemon API for richer gameplay.  
- **Multiple Rounds:** Supports multiple rounds for extended gameplay.  
- **Round Results Presentation:** Results presented in a **table format** after each round.  
- **Input Validation:** Ensures user inputs, like number of rounds or stat selection, are valid.  

**Skills Demonstrated:**
- Python programming, loops, functions, and data structures  
- API integration and JSON handling  
- Input validation and error handling  
- Basic game logic and UX considerations
